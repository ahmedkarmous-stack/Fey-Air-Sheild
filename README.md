# Fey Air Shield

Page web commerciale interactive pour presenter **Fey Air Shield**, un film solaire transparent concu pour reduire la chaleur, filtrer les UV et garder la luminosite naturelle des vitrages.

## Apercu

Ce projet est une page HTML autonome qui combine:

- une page d'accueil type annonce publicitaire
- des effets visuels animes autour du film solaire
- une section de decouverte du produit
- un prototype 3D interactif du film multicouche
- des sections science, thermique, prix et transparence
- un parcours vente et checkout
- un espace feedback utilisateur

## Fichier principal

```text
SmartFilm_Commercial_3D.html
```

La page fonctionne directement dans un navigateur. Aucun build n'est necessaire.

## Technologies utilisees

- HTML5
- CSS3
- JavaScript vanilla
- Three.js pour la scene 3D
- Canvas HTML pour les graphiques et le spectre lumineux
- Google Fonts

## Lancer le projet

Ouvrir simplement le fichier:

```text
SmartFilm_Commercial_3D.html
```

dans Chrome, Edge, Firefox ou un autre navigateur moderne.

> Note: la scene 3D utilise Three.js via CDN. Une connexion internet est donc necessaire pour afficher le modele 3D.

## Fonctionnalites

### Accueil publicitaire

La premiere section presente Fey Air Shield comme une annonce produit avec logo, statistiques clefs, animation solaire, panneau de verre lumineux et appels a l'action.

### Decouverte du produit

La section de decouverte explique le concept du film solaire et introduit le prototype 3D.

### Prototype 3D interactif

Le modele 3D permet de visualiser les couches du film:

- PET interieur
- couche active TiO2
- cadre mousse adhesive
- lame d'air piegee
- joint d'etancheite
- PET exterieur

Un slider permet de passer d'une vue compacte a une vue eclatee.

### Sections techniques

La page contient des explications sur:

- le filtrage UV
- la reduction thermique
- la lame d'air isolante
- la transparence visible
- la comparaison de prix

### Vente et checkout

Le checkout permet de:

- choisir un pack
- modifier la surface en m2
- ajouter une option de pose
- calculer automatiquement le total
- saisir les informations de commande

### Feedback

Un espace feedback permet aux visiteurs de laisser:

- une note de 1 a 5
- un type de retour
- un contact optionnel
- un commentaire

2. Ajouter `SmartFilm_Commercial_3D.html` et `README.md`.
3. Dans les parametres du repository, activer GitHub Pages.
4. Choisir la branche principale comme source.
5. Renommer le fichier HTML en `index.html` si vous voulez que la page s'ouvre automatiquement a la racine.
